# Milestone v0.4.0: Recursive Coordination Requirements Archive

**Status:** SHIPPED 2026-05-01
**Outcome:** 27/27 requirements complete
**Source:** `.planning/REQUIREMENTS.md`

## Completed Requirements

### DELEGATE — `delegate` decision shape on `coordinator`

- [x] **DELEGATE-01** — Coordinator agents can return decision `{ type: "delegate", protocol, intent, model?, budget? }` where `protocol` is `sequential`, `broadcast`, `shared`, or `coordinator`.
- [x] **DELEGATE-02** — Runtime executes a delegated decision as a full sub-run; sub-run `result` returns to the coordinator's next decision context. Defaults: child model = parent model if omitted; child budget = parent's remaining if omitted.
- [x] **DELEGATE-03** — Invalid `delegate` payload throws `DogpileError({ code: "invalid-configuration", detail.path })`.
- [x] **DELEGATE-04** — Nesting works to depth `maxDepth` (default 4); exceeding it throws `DogpileError({ code: "invalid-configuration" })`.

### TRACE — sub-run events and replay

- [x] **TRACE-01** — Parent trace includes a `subRun.started` event when a delegated run begins.
- [x] **TRACE-02** — Parent trace includes a `subRun.completed` event with the child's full JSON-serializable trace inline, or `subRun.failed` with the child error.
- [x] **TRACE-03** — `Dogpile.replay(parentTrace)` replays embedded child traces without re-executing children and returns identical output, accounting, and event sequence.
- [x] **TRACE-04** — `src/tests/event-schema.test.ts` and `src/tests/result-contract.test.ts` lock the new event shapes.

### BUDGET — cancellation, timeout, cost, floors

- [x] **BUDGET-01** — Parent abort propagates to all in-flight children; child runs surface `DogpileError({ code: "aborted" })`.
- [x] **BUDGET-02** — Parent `budget.timeoutMs` is a hard ceiling. Children get the parent's remaining time as their default; per-decision budget override is honored.
- [x] **BUDGET-03** — Parent cost and token totals equal parent's own provider calls plus all children, recursively.
- [x] **BUDGET-04** — Parent termination policies operate over parent-level events only. `minTurns` / `minRounds` floors apply per-protocol-instance and do not propagate.

### CONCURRENCY — bounded parallelism

- [x] **CONCURRENCY-01** — Coordinator config accepts `maxConcurrentChildren` (default 4); excess delegated children queue.
- [x] **CONCURRENCY-02** — Local providers clamp child concurrency to 1 and emit `subRun.concurrencyClamped`.

### PROVIDER — locality hint

- [x] **PROVIDER-01** — `ConfiguredModelProvider` accepts optional `locality?: "local" | "remote"`.
- [x] **PROVIDER-02** — `createOpenAICompatibleProvider` auto-sets local locality for loopback/RFC1918 hosts; caller-supplied locality overrides auto-detection.
- [x] **PROVIDER-03** — Invalid `locality` throws `DogpileError({ code: "invalid-configuration" })`.

### STREAM — child event bubbling

- [x] **STREAM-01** — `Dogpile.stream(parent)` emits child events wrapped with `parentRunIds` ancestry and child `runId`.
- [x] **STREAM-02** — Within a single child, event order is preserved; cross-child order is unspecified.
- [x] **STREAM-03** — `StreamHandle.cancel()` on the parent cancels all in-flight child streams.

### ERROR — child failure escalation

- [x] **ERROR-01** — A child `DogpileError` surfaces as `subRun.failed` into the coordinator's next decision context.
- [x] **ERROR-02** — If the parent terminates without final synthesis after an unhandled child failure, the parent throws the child's original `DogpileError`.
- [x] **ERROR-03** — Child timeouts surface as `provider-timeout`; parent timeouts surface as `aborted` with `detail.reason: "timeout"`.

### DOCS — discoverability

- [x] **DOCS-01** — `docs/recursive-coordination.md` documents the `delegate` decision, propagation rules, concurrency, locality, and trace embedding with a worked example.
- [x] **DOCS-02** — `examples/recursive-coordination/` is runnable using `createOpenAICompatibleProvider`.
- [x] **DOCS-03** — README "Choose Your Path" table points at `delegate`.
- [x] **DOCS-04** — `CHANGELOG.md` v0.4.0 lists all public-surface additions.
